Investigation Overview¶

In this Analysis for Ford Gobike dataset, my main focus was on Time (months), Trip duration and User types and the relation between these factors.

Dataset Overview¶

Ford GoBike System Dateset contains 2,506,983 bike rides info about individual rides made in a bike-sharing system covering the greater San Francisco Bay area through 2019. The dataset contains information about:

  • Trip duration: trip duration in seconds, start/end date/time for the bike rides.
  • Stations info: start/end station, name, geolocation (latitude/longitude)
  • Customer info: type of member (user_type)
  • Bike id
  • Bike Share for all Trip
  • Rental access method

Summary¶

Ford Gobike dataset analysis output:

  • The majority of trips durations around 10 mins and months with frequent rides are warm months such as July.
  • Subscribers of the service seem to use the service of bike rides more than the Customers.
  • subscribed user have higher number of trips than the customers specially during warm time.
  • Customer users are using the bikes for longer period of time.
